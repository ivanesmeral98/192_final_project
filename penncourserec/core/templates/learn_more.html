How it works

PennCourseReview is a recommender system that uses a content filtering approach to recommend CIS courses to students based off a selection of courses they've historically enjoyed. The language relevant to a course is encoded with pre-trained ELMo word-embeddings loaded using the fantastic Magnitude package. PennCourseReview's frontend is built using Bootstrap and JavaScript. The Bubble UI is rendered through PixiJS using a template provided by Mark Elphinstone-Hoadley here .
